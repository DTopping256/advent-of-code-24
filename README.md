# AdventOfCode24

Code written in Elixir and compiled using `mix`.

## Running the code

Requires Erlang and Elixir to be installed.

On the command line run:

```sh
cd advent_of_code_24
mix compile
iex.bat -S mix
```

Then in the integrated Elixir terminal run each days module:

```ex
iex> Day1.run()
{:ok, "the answer"}
```
